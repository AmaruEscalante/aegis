# Notebox Public Roadmap

Last updated: Q2 2024

This page shares what we're working on and what's coming next for Notebox. Items on this roadmap reflect our current thinking and are subject to change based on user feedback and engineering priorities.

---

## Recently Shipped ✓

- **Inline code blocks** — Format inline code with backticks, rendered in monospace
- **Folder nesting (unlimited depth)** — Organize notes in deeply nested folder hierarchies
- **Dark mode** — System-synced or manually toggled
- **CSV export** — Export any table within a note as a CSV file
- **iOS share sheet** — Share text or URLs to Notebox directly from Safari or other apps

---

## In Progress (Q2–Q3 2024)

### Collaborative editing
Real-time multi-user editing with cursor presence and conflict resolution. We're using an operational transformation approach (not CRDT) to keep the implementation simple and predictable.

**Status:** In engineering, targeting Q3 beta.

### Web clipper browser extension
One-click clipping of web pages, articles, and selected text from Chrome and Firefox. Clips preserve formatting and attach source URL.

**Status:** Firefox version in QA, Chrome version in engineering.

---

## Planned (Q4 2024)

- **API v2** — Public REST API for third-party integrations. Read/write notes, create folders, manage tags.
- **Zapier / Make integration** — Native connectors for workflow automation without code.
- **Handwriting recognition** — Transcribe handwritten notes from images (iPad Apple Pencil support first).
- **Advanced search filters** — Filter by date range, tag, folder, and content type in a single query.

---

## Under Consideration (No Timeline)

- Native Windows app
- End-to-end encryption (opt-in)
- Team billing and shared workspaces
- Audio notes and transcription
